# 📊 Amazon Sales Dashboard (Power BI)

## Project Overview
This project is an interactive Amazon Sales Dashboard developed using Microsoft Power BI. It provides insights into sales performance, courier status, product categories, fulfillment methods, sizes, and geographical distribution of orders.

## Dashboard Features
- 📈 Total Sales KPI
- 🚚 Courier Status Analysis
- 👕 Top 5 Product Categories
- 📦 Delivery Partner Analysis
- 🏢 Fulfillment Analysis
- 📏 Size-wise Analysis
- 🗺️ State-wise Orders
- 🌍 City-wise Orders
- 📅 Interactive Date Slicer

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Dataset
- **Source:** Kaggle
- **Dataset:** Amazon Sales Report

> This dashboard is created for educational and portfolio purposes using a publicly available dataset from Kaggle.
